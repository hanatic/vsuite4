# vsuite4
The Ultimate Productivity Network.

V Suite 4 is the fourth major release of V Suite by ViggyCorp Software, part of ViggyCorp UK.

Socials:	
	
	Twitter: @viggycorp_s  @viggycorp

V Suite 4 is the first major release to run independently from Anvil, a platform allowing fast web development with only Python. This new version runs more
independently and is more advanced than previous versions. This is the biggest upgrade yet in appearance of the UI and responsiveness as we move to a new method
of development.
